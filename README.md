# Лабораторная работа №5. Авторизация и статистические отчёты

Веб-приложение для управления учётными записями пользователей с авторизацией на основе ролей и журналом посещений.

## Функциональность

### Авторизация (роли)

**Администратор:**
- Создание/редактирование/удаление любых пользователей
- Просмотр профилей всех пользователей
- Доступ к полному журналу посещений

**Пользователь:**
- Редактирование только своих данных (роль недоступна для изменения)
- Просмотр только своего профиля
- Доступ к журналу посещений только своих записей

### Журнал посещений
- Таблица visit_logs с пагинацией
- Статистика по страницам с экспортом в CSV
- Статистика по пользователям с экспортом в CSV

## Запуск локально

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Приложение: http://127.0.0.1:5000
